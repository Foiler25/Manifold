**Every port, mapped.**

A free, open-source macOS app that visualizes physical USB and Thunderbolt connections in real time, with per-port power draw and link-speed telemetry. Menu bar utility and standalone window app in one.

> **WIP — pre-alpha.** Phase 0 scaffolding only. Real functionality lands phase by phase; see `BUILD_LOG.md` for the running record.
